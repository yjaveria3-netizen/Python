## 🚀 Getting Started

### Running Examples
```bash
# Clone or download the examples
git clone [repository-url]
cd docs/examples

# Run a specific example
python beginner/hello_world.py

# Run with virtual environment
python -m venv example_env
source example_env/bin/activate  # macOS/Linux
# example_env\Scripts\activate   # Windows
pip install -r requirements.txt
python example_file.py
```

### Prerequisites
- Python 3.8+ installed
- Basic understanding of Python syntax
- Code editor or IDE
- Command line familiarity

### Required Packages
```bash
# Common packages for examples
pip install requests pandas numpy matplotlib
pip install flask django fastapi
pip install pytest black flake8
```
